## TYPE

| 타입     | 설명                                 |
| -------- | ------------------------------------ |
| feat     | 새로운 기능 추가 구현                |
| init     | 프로젝트 초기 설정                   |
| fix      | 에러 수정                            |
| refactor | 코드 리팩토링                        |
| docs     | 문서 작성                            |
| update   | 이름 수정 , 파일 삭제 등 전체에 적용 |





## ISSUE

### [PART] FUNCTION

PART : USER(회원), BOARD(게시판), DIST(배포)
